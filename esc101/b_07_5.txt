                <h1><span class="title">Robber in Dilemma</span><br><small class="category">Bonus7</small></h1>
                <hr>
                <div style="white-space: pre-wrap" class="statement">You, along with Alice and Bob, are planning to rob the SnT council bank. In the bank, there are n bags of gold coins. You want to distribute the bags among the 3 robbers, in such a way, that each robber gets an equal number of coins. You cannot transfer a coin from one bag to the other.
Formally, you will be provided with the count of coins in these n bags, you need to answer with "YES" or "NO", whether you can split the bags among the 3 robbers, such that each of you has an equal number of gold coins. You need to solve this problem for multiple test cases.

<b>NOTE:</b> You need to solve this problem using recursion.

<b>INPUT</b>:
First line of the input contains a single integer t - denoting the number of test cases you need to solve. Next, you have the description of these t test cases. First line of each test case contains n - denoting the number of bags in SnT bank. Second line of each test case contains n space-separated integers, denoting the count of gold coins in these n bags.

<b>Sample Input</b>
2
6
1 2 2 2 2 3
3
1 1 2

YES
NO

<b>Explanation:</b>
In the first test case there are 6 bags, we can distribute them as: Alice gets {2,2}, Bob gets {2,2}, and you get {1,3}. Since all of you have equal gold coins, partitioning is possible.
In the second test case, there is no possible way.</div>
